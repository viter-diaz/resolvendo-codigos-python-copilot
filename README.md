# Resolvendo Códigos em Python com GitHub Copilot

Este projeto foi desenvolvido como parte do desafio da DIO: **Utilizando as Ferramentas do GitHub para Solucionar Algoritmos em Python**.

## Objetivo

O objetivo do projeto é praticar a resolução de algoritmos em Python utilizando o apoio do GitHub Copilot, explorando sugestões de código, melhoria de lógica e produtividade no desenvolvimento.

## Tecnologias utilizadas

* Python
* Git
* GitHub
* GitHub Copilot
* Markdown

## Estrutura do projeto

```txt
resolvendo-codigos-python-copilot/
│
├── desafios/
│   ├── desafio_01.py
│   ├── desafio_02.py
│   └── desafio_03.py
│
└── README.md
```

## Desafios implementados

### Desafio 01

Verificador de número par ou ímpar.

### Desafio 02

Calculadora simples com operações básicas.

### Desafio 03

Outros algoritmos em Python para praticar lógica de programação.

## Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Criação de repositório no GitHub;
* Versionamento de código com Git;
* Escrita de documentação com Markdown;
* Resolução de problemas usando Python;
* Uso do GitHub Copilot como apoio no desenvolvimento;
* Organização de projeto para portfólio.
